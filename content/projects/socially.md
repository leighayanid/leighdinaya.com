Just another linktr.ee clone.

Store all your social links in one place. Add your social media links, music playlist links, payment links, blogs etc.

Powered by Nuxt.js and Supabase.

Note: This project is still in development.

## App Features

- User Authentication
- Add unlimited links
- Shareable links
- Profile Management

## Todo Features

- Stats
- Themes

## Tech Stack

### Client | Front-end

- Nuxt.js 2 (SSR)
- TailwindCSS
- DaisyUI

### Server | Backend

- Vercel
- Supabase

## Supabase Features Used

- Supabase Auth
- Supabase Database
- Supabase Storage
